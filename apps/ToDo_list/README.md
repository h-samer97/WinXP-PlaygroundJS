# Task Manager UI

A simple Task Manager built with vanilla JavaScript, HTML, and CSS.  
It allows you to add, delete, and mark tasks as completed, while persisting data in `localStorage`.

## Features

- Add tasks via input field and button
- Delete tasks with a single click
- Mark tasks as finished by toggling their state
- Persistent storage using `localStorage`
- Task counters for total and completed tasks
- Empty state message when no tasks are available
